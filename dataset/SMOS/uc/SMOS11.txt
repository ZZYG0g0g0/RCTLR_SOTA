
Name: Assign / Remove Teachings To Teacher
Actors: Administrator
Description: Allows you to assign or remove courses to a teacher
Preconditions:
• The user is logged in as an administrator • The user has carried out the use case "View User Details" and the system is
viewing the details of a teacher • The user clicks on the "Teacher Teachings" button
Sequence of events
User
2. Select the academic year
4. Select the desired class
6. Select the courses to associate or remove with the teacher
System
1. View the teaching management form for the teacher in question
3. Displays the list of classes available for the selected year
5. View the list of courses associated with that class
7. Assign or remove teacher's teachings as indicated by the Administrator
Postconditions:
• • •
One or more teachings have been assigned or removed to the teacher The administrator interrupts the operation Connection to the SMOS server interrupted
