
Name: InsertNewClass
Actors: Administrator
Description: Adds a new class to the archive
Preconditions:
• The user is logged into the system as Administrator • The user has already run the use case "View Class List" and the system
is viewing the list of classes for a particular academic year. • The user clicks on the "New Class" button
Sequence of events
User
2. Fill out the form 3. Click on the "Save" button
System
1. The system shows the form to be filled in with: name, address, academic year.
4. Checks the validity of the data entered and enters the new class in the archive; in the event that the data entered is invalid, activate the "Data Error" use case.
Postconditions:
• • • •
The user has entered a class in the system The data error is notified Connection to the SMOS server interrupted The Administrator interrupts the operation
