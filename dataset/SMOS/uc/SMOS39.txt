Name: ViewNoteDetails
Actors: Administrator
Description: Displays the details of a note
Preconditions:
• • •
The user must be logged in to the system as Administrator The user has carried out the use case "ViewNote List" Click on the "details" button associated with one of the notes
Sequence of events
User
System
1. Show a form with the details of the note: pupil, description, teacher, date.
Postconditions:
• • •
The system is displaying the details of a note. The administrator interrupts the operation Connection to the SMOS server interrupted
